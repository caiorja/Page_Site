# Crainf - Business Intelligence

Site oficial da Crainf - Consultoria em Business Intelligence, ETL e Data Analytics.

## Sobre

A Crainf nasceu com o propósito de transformar dados em clareza estratégica. Combinando tecnologia, inteligência de negócios e metodologias como o Six Sigma, desenvolve soluções que ajudam empresas a compreender processos, medir desempenho e tomar decisões assertivas.

## Tecnologias

- HTML5
- CSS3
- JavaScript
- Design Responsivo

## Deploy

Este site é hospedado automaticamente no GitHub Pages a partir da branch `main`.
